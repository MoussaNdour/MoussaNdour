About Me

Je suis un étudiant en licence informatique passionné par le développement et les mathématiques. Toujours curieux et avide d’apprendre, je me consacre à maîtriser des domaines variés tels que le développement web, mobile et logiciel, avec une ambition claire : devenir un développeur accompli.

Mon parcours est guidé par une volonté de résoudre des problèmes complexes et de concevoir des solutions innovantes.

En parallèle, je cultive une passion pour la physique, que je considère comme un pont entre les mathématiques et l’exploration de notre univers. Mon objectif à long terme est de me spécialiser en informatique quantique, un domaine où se croisent mes intérêts pour la science et la technologie.

À travers ce portfolio, je partage mes projets, mon évolution et ma vision, tout en me préparant à relever les défis du monde technologique.

Langages Appris

HTML et CSS

Mon experience avec HTML et CSS

Je possède une bonne maîtrise de HTML et CSS, ce qui me permet de structurer et de styliser des interfaces web de manière efficace. Je suis à l’aise avec les balises HTML sémantiques, la gestion des formulaires, les tableaux, et les éléments multimédias. Côté CSS, je sais utiliser les propriétés de mise en page (Flexbox, Grid), gérer les animations, et créer des designs responsives adaptés aux différents écrans. Mon objectif est de produire des interfaces claires, modernes et accessibles.

Javascript

Mon experience avec Javascript

En JavaScript, je suis capable de concevoir des fonctionnalités dynamiques pour enrichir l’expérience utilisateur. Je maîtrise les bases du langage, telles que la manipulation du DOM, la gestion des événements, et l’utilisation des API natives. Je suis également familier avec des concepts plus avancés, comme les promesses, la programmation asynchrone, et les modules ES6. Mon approche consiste à écrire un code lisible, optimisé, et facile à maintenir pour répondre aux besoins des projets.

Bootstrap

Mon experience avec Bootstrap

Je maîtrise les bases de Bootstrap, l’un des frameworks CSS les plus populaires pour concevoir des interfaces web modernes et réactives. Avec Bootstrap, je peux créer des sites qui s'adaptent parfaitement à toutes les tailles d’écran. Ses composants prédéfinis et son système de grille facilitent le développement rapide de designs professionnels, sans avoir à écrire beaucoup de code CSS personnalisé.

PHP

Mon experience avec PHP

J’ai appris les bases de PHP, un langage côté serveur indispensable pour le développement web. Grâce à PHP, je peux créer des applications dynamiques et interactives, gérer des bases de données et automatiser des tâches côté serveur. Ce langage est puissant pour la création de systèmes comme les blogs, les plateformes e-commerce ou les outils de gestion. Maîtriser PHP est un atout essentiel pour tout développeur web.

SQL

Mon experience avec SQL

J’ai acquis les bases de SQL, le langage standard pour interagir avec les bases de données relationnelles. Avec SQL, je peux créer, gérer et interroger des bases de données pour stocker et exploiter efficacement des informations. Cela me permet de travailler avec des systèmes comme MySQL et d’intégrer des données dans des applications web ou logicielles.

C++

Mon experience avec C++

Je me suis initié aux bases de C++, un langage puissant et polyvalent utilisé pour le développement de logiciels performants. C++ m’a permis de comprendre les concepts fondamentaux de la programmation comme la gestion de la mémoire, la programmation orientée objet et la manipulation de structures de données. C’est une excellente base pour explorer des domaines comme le développement de jeux, les systèmes embarqués ou encore l’intelligence artificielle.

React

Mon experience avec React

J’ai appris les bases de React, une bibliothèque JavaScript incontournable pour créer des interfaces utilisateur dynamiques et réactives. Grâce à React, je peux développer des applications web modernes avec des composants réutilisables et une gestion efficace de l’état. Cette compétence est essentielle pour répondre aux besoins des utilisateurs en matière de performance et d’interactivité.
